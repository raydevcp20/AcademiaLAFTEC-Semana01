Git e Github

Git

O git é um sistema de controle de versão mais ultilizado recentemente, seu objetivo é deixar optimizada as versões de desenvolvimento de qualquer código. Tendo como criador Linus Torvalds, criador do sistema operacional Linux, o Git possui código aberto e tem manutenção constante desde 2005.

Github

O github é um serviço para hospedagem de versionamento de códigos git, ou seja, ele serve para a hospedagem dos repositórios criados a partir do git, principalmente utilizada por desenvolvedores para expor códigos e compartilhar eles com outros desenvolvedores além de apresentar uma forma fácil e prática para apresentar versões de projetos que podem ser atualizadas com o tempo.

Principais comandos

Iniciar um repositório-
git init

"Baixar" um repositório-
git clone link_do_repositório

Puxar alterações do repositório-
git pull

Listando o caminho do repositório-
git remote -v

Adicionando um arquivo para subir no github-
git add nome_do_arquivo

Commitar (salvar as alterações) para uma branch-
git commit -m "mensagem"

Enviando uma branch para o repositório-
git push origin nome_da_branch
